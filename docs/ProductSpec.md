## Positioning
A privacy-first app that prevents accidental exposure of sensitive conversations and files in shared environments (cars, meetings, shared devices) by keeping them in an encrypted private workspace.

## Core Modules
1) Cover Mode (user-selected theme)
2) Secret Entrance (unlock)
3) Private Room dashboard
4) Private Circle (contacts)
5) Secure Messenger (E2EE)
6) Vault (E2EE local storage; optional cloud backup)
7) Privacy Modes (Car/Work/Public) + panic return

## Platform support
- iOS: Secure Messenger + Vault + Privacy Modes (Shortcuts automation)
- Android: Same + optional Default SMS Mode (public inbox + private inbox)

## MVP features (ship)
- E2EE 1:1 chats + attachments
- Private vault with import/share-sheet
- Generic/silent notifications
- Auto-lock + app switcher blur
- Cover Mode + secret unlock
- Subscription paywall for premium features

## V2 features
- Group chats (Sender Keys)
- Multi-persona workspaces
- Sealed sender / metadata reduction
- Encrypted cloud backups
- Safety-number verification improvements
